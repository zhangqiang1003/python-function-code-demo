### [多继承的mro](./多继承的mro.py)
![](http://jbcdn2.b0.upaiyun.com/2016/07/cffe957b3b29bd1e0877afe8fae5105e.jpg)

* 解释
1. 找没被指向的类，放入集合中，并且删除A指向的边，mro = [A]
2. 找没被指向的类，此时有BC两个类，按照左边优先的原则，将B放入集合，并删除B指向的边，mro = [A，B]
3. 重复2过程，如果发生多个类无法确定左右关系的，下级优先。
